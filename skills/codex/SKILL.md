# Codex連携スキル

## 概要

OpenAI Codex CLIを使用して、コードレビュー・バグ調査・設計相談・文言検討などを別の視点から分析します。

## 実行方法

以下のコマンドを実行してください：

```bash
codex exec --full-auto --sandbox read-only --cd "!`pwd`" "$ARGUMENTS"
```

## パラメータ説明

- `--full-auto`: 完全自動実行モード
- `--sandbox read-only`: 読み取り専用の安全な分析環境
- `--cd`: 現在のプロジェクトディレクトリを指定

## 使用例

- `/codex このコードをレビューして`
- `/codex バグの原因を調査して`
- `/codex この設計について意見を聞かせて`
- `/codex エラーメッセージの文言を検討して`

## 注意事項

- Codexの実行には時間がかかる場合があります
- 結果はターミナル出力で確認できます
- 読み取り専用モードのため、ファイルの変更は行われません

## 結果の活用

Codexからの回答を受け取ったら、その内容を要約してユーザーに報告してください。
必要に応じて、Codexの提案を実装に反映することを検討してください。
